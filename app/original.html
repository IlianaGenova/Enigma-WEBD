THE
ENIGMA
COMPONENTS

LAMP
BOARD

The Enigma's lampboard component is displayed up above on the banner picture. This is the output of the machine and displays the corresponding letter, when an input is provided with the right settings

KEY
BOARD

The Enigma's keyboard component is displayed up above on the banner picture. This is used for input if the machine. The German soldiers used this to enter the coded messages, and when the right settings were set, the message was decoded correctly

PLUG
BOARD

The Enigma's plugboard component is displayed up above on the banner picture. The plugboard was the additional encoding added later on which provides a lot more complexity to the cipher. Connecting the plugs routes the letters - so if A is connected with B, A follows the path of the letter B in encoding

CORE
BOX

The Enigma's rotors component is displayed up above on the banner picture. This is the core of the machine. Every letter is intertwined with another in these three components, so A never returns an A. The reflector provides four level deep encoding.
